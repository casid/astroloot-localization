Boon

This module contains all Boon titles. Boons are items granted to the player for beating a certain boss enemy in the game.

The titles in this module are shown when choosing 1 of 3 options for a given Boon. See screenshot BOON_SELECTION.

The name of the boon item itself is part of the ITEM module. 
The attributes of each Boon option are part of the ATTRIBUTE module.
