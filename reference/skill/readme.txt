Skill

This module contains all skill names on the skill tree.

The various effects a skill can have are located in the ATTRIBUTE module.

Skill info layout (see screenshot SKILL_INFO):
- Name of the skill (located in this module)
- Attributes of the skill (located in ATTRIBUTE module)
- Optional skill diff (if skill allocation would change stats, located in COMMON)


Players can press alt (or a controller key) to show details about a skill. Then all Glossary search terms found become underlined and can be hovered for a description of the term (see GLOSSARY).
